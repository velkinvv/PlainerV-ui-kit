# Интеграция аккордиона с макетом Figma

## Обзор

Аккордион был полностью переработан согласно макету из Figma: [Проект](https://www.figma.com/design/DxSeBfd1bBXCF4DlsIeLUr/%D0%9F%D1%80%D0%BE%D0%B5%D0%BA%D1%82?node-id=4873-15261&t=uGsIilm1pmzcb7f3-4)

## Основные изменения

### 1. Логика множественных элементов

- ✅ **Исправлена логика**: теперь открывается только тот элемент, на который нажали
- ✅ **Поддержка allowMultiple**: возможность открывать несколько элементов одновременно
- ✅ **Уникальные ID**: каждый элемент имеет свой уникальный идентификатор

### 2. Дизайн согласно макету Figma

- ✅ **Заголовок и подзаголовок**: основной заголовок 16px Medium + подзаголовок 12px Regular
- ✅ **Позиционирование**: поддержка позиций start/center/last с разными радиусами
- ✅ **Границы**: правильные границы между элементами (#ECECEC)
- ✅ **Иконки**: стрелка влево для закрытого состояния, стрелка вниз для открытого

### 3. Использование переменных темы

- ✅ **Цвета**: используются переменные из темы для соответствия макету Figma
- ✅ **Шрифты**: используются переменные fonts.primary из темы
- ✅ **Структура**: убран общий контейнер, каждый элемент независим
- ✅ **Границы**: правильные границы между элементами

### Цвета (через переменные темы)

- **Фон**: `theme.colors.backgroundTertiary` (#f5f5f5)
- **Текст заголовка**: `theme.colors.text` (#424242)
- **Текст контента**: `theme.colors.textQuaternary` (#757575)
- **Hover фон**: `theme.colors.backgroundQuaternary` (#f0f0f0)
- **Границы**: `#ECECEC` (Gray/100) - оставлен хардкод для точного соответствия макету

### Типографика

- **Шрифт**: `theme.fonts.primary` (Montserrat)
- **Заголовок**: 16px, Medium (500)
- **Подзаголовок**: 12px, Regular (400)
- **Контент**: 14px, Regular (400)

### Отступы и размеры

- **Заголовок**: `8px 16px`
- **Контент**: `4px 16px 16px`
- **Радиус**: `8px` (точный из макета)
- **Иконка**: `16x16px`
- **Gap между элементами**: `4px`

### Позиционирование элементов

- **Start**: `border-radius: 8px 8px 0px 0px`
- **Center**: `border-radius: 0`
- **Last**: `border-radius: 0px 0px 8px 8px`

## Новые возможности

### API компонента

```tsx
<Accordion allowMultiple={false}>
  <Accordion.Item value="item1" position="start">
    <Accordion.Trigger title="Заголовок" subtitle="Подзаголовок" />
    <Accordion.Content>Контент</Accordion.Content>
  </Accordion.Item>
</Accordion>
```

### Пропсы

- `allowMultiple`: позволяет открывать несколько элементов одновременно
- `value`: уникальный идентификатор элемента
- `position`: позиция элемента (start/center/last)
- `title`: основной заголовок
- `subtitle`: подзаголовок
