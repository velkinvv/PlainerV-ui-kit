### 8. Версионирование

**Проблема:** Версия 1.0.0 при отсутствии тестов

**Шаги:**
- ⬜ Изменить версию на `0.1.0` (pre-release)
- ⬜ Установить semantic-release:
  ```bash
  npm install --save-dev semantic-release @semantic-release/changelog @semantic-release/git
  ```
- ⬜ Создать `.releaserc.json`:
  ```json
  {
    "branches": ["main"],
    "plugins": [
      "@semantic-release/commit-angular",
      "@semantic-release/release-notes-generator",
      "@semantic-release/changelog",
      "@semantic-release/npm",
      "@semantic-release/git"
    ]
  }
  ```
- ⬜ Настроить commitlint для conventional commits
- ⬜ Обновить скрипты в package.json:
  ```json
  "release": "semantic-release"
  ```
- ⬜ Протестировать локально

**Ожидаемый результат:** Автоматическое версионирование на основе коммитов
